# 飞控层说明（ArduSub/PX4 参考实现）

本目录用于管理底层飞控配置与硬件映射。

## 建议硬件

- 飞控：Pixhawk 4/6 系列
- IMU/磁罗盘：飞控内置 + 外置罗盘（可选）
- 深度计：MS5837 或同类
- 推进器：6~8 推进器布局（X/矢量）

## 通信建议

- 主控机（NVIDIA Jetson / x86）与飞控：
  - 串口：`/dev/ttyACM0`（921600）
  - 或 UDP：`udp://0.0.0.0:14550`
- 协议：MAVLink v2

## 参数模板

示例参数见 `params/ardusub_example.params`。
